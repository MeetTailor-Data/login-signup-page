# Login–Signup Page

##  Project Overview

This project is a **Login and Signup Page UI** designed for general-purpose authentication systems. It is **not dedicated to any specific domain** (such as a library, college, or company) and can be reused in different projects with minimal changes.

The page includes a clean, modern interface with tab-based switching between **Login** and **Signup** forms.

---

##  Features

* Login form with ID and password fields
* Signup form with user details and password confirmation
* Tab-based switching (Login / Signup)
* Guest login option (redirects to home page)
* Fully responsive design
* Modern UI using CSS gradients

---

##  Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Google Fonts (Poppins)

##  How to Use

1. Clone or download the repository
2. Open `index.html` in any modern web browser
3. Use the Login or Signup tabs to interact with the UI

##  Customization

You can easily:

* Connect this UI to any backend (PHP, Node.js, Firebase, etc.)
* Rename labels to match your project (Admin, User, Student, etc.)
* Separate CSS and JavaScript into external files
